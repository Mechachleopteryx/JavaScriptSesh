JavascriptSesh demo (v1)

A test of the MdC 2 syntax and display. First, we simply try Javascript libraries for a good display with scaling. This is a very simple demo of the use of Javascript JSesh. It's completely alpha software now. Our system seems to have a small problem with firefox, BTW (but eventually the positioning system will be completely different, so we don't care too much).

The sentence i-w-r:a-C1-m-p*t:pt means "Re is in the sky"

We support only very simple Manuel de codage Basically, you can write simple groups (with parenthesis), and also // for shaded parts, [[ and ]] for lacunas, and ".." for spaces. Note that "-" are mandatory between groups. pA-x:r:xr-[[-..-]]-n:(x:t)*U30-xAst-// has code pA-x:r:xr-[[-..-]]-n:(x:t)*U30-xAst-//

Use in a HTML Table:
Signcode
A1A1
A2A2

JavascriptSesh parser built using PEGJS.
